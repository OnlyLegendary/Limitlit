# LimitLit

**LimitLit** is a deliberately limited software renderer inspired by the creative spirit of 1990s rendering tools.

The goal is not to become Blender. The goal is to make a focused renderer where constraints spark creativity.

## v0.1.0

This version includes:

- SDL2 application window
- Application class
- Window class
- Renderer class
- Framebuffer class
- Color class
- Software framebuffer presented to the screen
- Classic gray background generated by LimitLit itself

## Build on Ubuntu

Install dependencies:

```bash
sudo apt update
sudo apt install cmake g++ pkg-config libsdl2-dev
```

Build:

```bash
mkdir -p build
cd build
cmake ..
make
./LimitLit
```

## Philosophy

LimitLit should stay simple, readable, and creative.

Rules:

- No node editor
- No plugin system
- No giant hidden settings menus
- Renderer owns the pixels
- SDL only creates the window and displays the framebuffer
- Every feature should have a clear creative purpose
