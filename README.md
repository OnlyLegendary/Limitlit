# LimitLit

A deliberately limited ray tracer inspired by the creative simplicity of 1990s rendering software, Inspired by the looks and feels of bryce2


LimitLit embraces constraints.

Instead of adding every feature imaginable, it provides a carefully chosen set of tools that encourage creativity.

- Fast startup
- Simple interface
- CPU ray tracing
- Cross platform
- No plugins
- No scripting
- No node editors
