# LimitLit Vision

LimitLit is a deliberately constrained software renderer.

It is inspired by the feeling of late-1990s creative rendering tools: simple interfaces, visible controls, and a workflow that encourages experimentation instead of endless configuration.

## What LimitLit Is

- A CPU software renderer
- A creative renderer
- A small, understandable codebase
- A project where one person can understand the whole engine

## What LimitLit Is Not

- Not a Blender replacement
- Not a game engine
- Not a physically perfect renderer
- Not a giant plugin platform
- Not an AI rendering tool

## Core Principle

Boundaries create creativity.
